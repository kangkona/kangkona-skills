# Kangkona Skills

> 面向 builders、operators 和 indie hackers 的可复用实战 playbooks，帮助你在真实约束下做出更锋利的判断，并交付可信成果。

语言：[English](README.md) | 中文

## 安装

安装完整 skill collection：

```bash
npx skills add kangkona/kangkona-skills
```

然后在支持 skills 的助手中调用 hackathon playbook：

```text
Use $hackathon 帮我选择黑客松项目、打磨叙事、规划执行，并准备 demo/PPT/Q&A。
```

## 这些 Skills 的不同之处

- **实战验证**：来自多次黑客松和 builder 现场经验，不是抽象建议。
- **评分表优先**：先看评分标准、截止时间、受众、赛道和提交物，再选择项目。
- **规则意识**：显式检查原创性、既有代码、外部依赖、API、数据集和提交约束。
- **Demo 导向**：优先打磨一个可见、可信、端到端的闭环，而不是堆一组脆弱功能。
- **双语协作**：内容支持英文和中文协作场景。

## 当前集合

### Hackathon / 黑客松

当你需要做这些事时，可以使用这个 skill：

- 在多个黑客松项目想法之间做选择。
- 把模糊想法压缩成清晰的一句话、品类和 demo loop。
- 在不太早恋战的前提下，对多个候选项目做小规模赛马。
- 构建最小可信 demo，并准备截图、兜底路径和难以伪造的证明。
- 准备短 pitch、Q&A 防守包和最后提交 checklist。
